In addition to the default WordPress sources, EmbedPress supports these providers:

- 23h (Images)
- AmCharts (Charts)
- Bambuser (Videos)
- Cacoo (Charts)
- ChartBlocks (Charts)
- Chirbit (Audios)
- Cly (Audios)
- CircuitLab (Charts)
- Coub (Videos)
- Crowd Ranking (Polls)
- Daily Mile (Activity)
- Devianart (Images)
- Dipity (Timelines)
- Dotsub (Videos)
- Edocr (Documents)
- Facebook (Posts)
- GettyImages (Images)
- Github Gist (Code)
- Google Docs (Documents)
- Google Maps (Maps)
- Google Drawings (Drawings)
- Google Sheets (Spreadsheets)
- Google Slides (Presentation Slideshows)
- HuffDuffer (Audios)
- Infogram (Charts)
- MobyPicture (Image)
- NFB (Videos)
- Porfolium (Projects)
- Release Wire (Press releases)
- Roomshare (Listings, in Japanese)
- Sapo Videos (Videos, in Spanish)
- ShortNote (Notes, in Japanese)
- Rutube (Videos, in Russian)
- Shoudio (Audios)
- Sketchfab (Drawings)
- Ustream (Videos)
- Viddler (Videos)
- VideoJug (Videos)
